# RedAudit Documentation

**RedAudit** is an automated network auditing framework that orchestrates industry-standard security tools (`nmap`, `nikto`, `testssl.sh`, `nuclei`) into a concurrent pipeline. It is designed for **defensive hardening** and **authorized offensive assessments**, focusing on secure, deterministic, and structured output.
