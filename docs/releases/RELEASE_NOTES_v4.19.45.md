# RedAudit v4.19.45 - Immediate Startup Update Notice

## Summary

This patch improves startup update awareness so users see new releases as soon as they open RedAudit.

## Added

- No new user-facing features were added in this release.

## Improved

- Startup update checks are now performed on every launch (non-blocking, short timeout).

## Fixed

- Startup update checks now use cached release metadata as fallback when the network is unavailable.

## Testing

- Internal validation completed.

## Upgrade

- No action required.
