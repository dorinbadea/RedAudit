# RedAudit v4.19.48 - Gestion de Entradas de Reanudacion Nuclei

[![View in English](https://img.shields.io/badge/View_in_English-blue?style=flat-square)](https://github.com/dorinbadea/RedAudit/blob/v4.19.48/docs/releases/RELEASE_NOTES_v4.19.48.md)

## Resumen

Este parche mejora la usabilidad en ejecuciones parciales repetidas de Nuclei al incorporar limpieza integrada de reanudaciones en el flujo interactivo.

## Anadido

- El menu **Reanudar Nuclei (pendiente)** ahora incluye **Gestionar entradas de reanudacion**.
- Puedes borrar una entrada antigua concreta o limpiar todas las entradas pendientes directamente desde el wizard.

## Mejorado

- El mantenimiento de reanudaciones ya no depende de borrar ficheros manualmente en carpetas de reportes.
- La documentacion ahora refleja este flujo de limpieza desde menu en README, uso, manual y guia didactica.
