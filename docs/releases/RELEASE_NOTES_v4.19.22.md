# RedAudit v4.19.22 - Wizard Back Navigation and Long Scan Warning

## Summary

Adds back/cancel navigation to Standard and Exhaustive profile prompts and warns when long runtimes are likely.

## Added

- None.

## Improved

- None.

## Fixed

- Wizard now allows back/cancel in Standard and Exhaustive profile prompts.
- Full or Nuclei-enabled modes now warn about runs exceeding 4 hours.

No breaking changes. Update to v4.19.22.
