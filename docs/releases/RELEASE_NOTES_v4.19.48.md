# RedAudit v4.19.48 - Nuclei Resume Entry Management

[![Ver en Español](https://img.shields.io/badge/Ver_en_Español-red?style=flat-square)](https://github.com/dorinbadea/RedAudit/blob/v4.19.48/docs/releases/RELEASE_NOTES_v4.19.48_ES.md)

## Summary

This patch improves day-to-day usability for repeated Nuclei partial runs by adding built-in resume-entry cleanup in the interactive flow.

## Added

- The **Resume Nuclei (pending)** menu now includes **Manage resume entries**.
- You can delete a single stale resume entry or clear all pending resume entries directly from the wizard.

## Improved

- Resume maintenance no longer depends on manual file cleanup in report folders.
- Documentation now reflects the in-menu cleanup workflow across README, usage, manual, and didactic guides.

## Fixed

- No additional bug fixes in this release.

## Testing

- Internal validation completed.

## Upgrade

- No action required.
